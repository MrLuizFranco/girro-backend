# G!rro Backend

Este é o backend do aplicativo de mobilidade urbana **G!rro**, feito em Node.js com banco de dados SQLite e hospedagem do arquivo `.db` no GitHub via Firebase Raw.

## Tecnologias usadas

- Node.js
- Express
- SQLite3
- Axios
- Firebase (para hospedar o banco)
- Dotenv
